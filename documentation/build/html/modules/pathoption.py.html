pathoption¶

Utility module for the option “–path” for ncp_paths_<software>.py scripts.

The function create_pathoption returns an PathOption object.

The function register_pathoption registers a new path option permitted value and the corresponding function reference.

pathoption.create_pathoption()¶

Returns a PathOption object

rtype:

PathOption


pathoption.register_pathoption(pathoption, pathkey, function_reference)¶

Registers data pair pathkey:function_reference with object pathoption.

Parameters:

  • pathoption (PathOption) – The object to register the data pair.

  • pathkey (str) – The –path option value

  • function_reference – A reference to a function


class pathoption.PathOption¶

PathOption stores the function references that when invoked return the installation paths. The function references are stored in data pairs pathkey:function_reference where pathkey corresponds to the –path option value.

add_option(self, pathkey, ref_to_function)¶

Adds the data value pair pathkey:function_reference. If a duplicate pathkey value an exception will be raised.

Parameters:

  • pathkey (str) – A –path option value

  • ref_to_function – A function reference that when invoked returns a string.

Raises:

ErrorDuplicatePathOptionKey – If an existing pathkey is already registered

get_path(self, key)¶

For the value of key, returns the corresponding function reference.

Parameters:

key (str) – A –path option value

Return type:

str

Returns:

The corresponding path for the key value.
